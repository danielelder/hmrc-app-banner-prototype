Web app banners

If this page is viewed on Android or iOS an install banner should be displayed.

This is a JavaScript solution created as a proof of concept for an Optimizely experiment.

How to test

Using Chrome browser open the developer tools and toggle the device toolbar on.

Select either an iOS (iPhone SE) or Android (Samsung Galaxy S8+) and then refresh the browser.

Actions

Closing the banner should add a cookie called smartbanner_exited, this will prevent it from being shown for 30 days.

You can change the delay using the meta tag smartbanner:hide-ttl.
